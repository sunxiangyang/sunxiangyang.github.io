# 查看正在执行的sql

# greenplum查看集群状态
gpstate -e

# greenplum集群恢复

## 主备对调
gprecoverseg -r

# 强制恢复集群
gprecoverseg -F

#greenplum启动重启关闭
gpstart 
 常用参数  -a 无需终端输入yes确认
          -m 只启动master节点
gpstop
常用参数   -u 不停止数据库 只重新加载配置参数如pg_hba_conf、postgres.conf
          -f 强制停止数据库
          -r 重启数据库
